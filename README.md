# XZ Token

## Initial supply

- 21 million

## Manage token client

- `client_add` Add new client
- `client_remove` Remove added client
- `client_show` Show clients list
- `client_zero` Zero client spread
- `client_spread` Show client spread

## Burn/Mint Process

- Burn: Client wins
- Mint: Client loses

## Defense Mechanisms

Client can't exceed mint spread more than 500,000.
